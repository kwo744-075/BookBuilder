## Requirements
- **Internet connection** — voices stream from the Microsoft Edge TTS service.
- **PDF input** also needs `pdftotext` (poppler-utils) installed on the system
  [Linux/macOS]. Other formats need nothing extra.

## Install
1. Download and unzip `BookBuilder-1.0.0-linux.zip`.
2. Open the `BookBuilder` folder and run the **BookBuilder** executable.

## Supported input formats
- `.txt`, `.pdf`, `.docx`, `.epub`

## Known limitations
- Scanned / image-only PDFs have no text layer to read.
