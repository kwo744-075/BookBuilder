# BookBuilder

## Mission

BookBuilder is a one-click audiobook creator that converts books into organized MP3 audiobooks using high-quality AI voices.

## Current Status

Version: 0.1 Stable

## Supported Input

* TXT ✅
* EPUB (planned)
* PDF (planned)
* DOCX (planned)
* MOBI (planned)
* AZW3 (planned)

## Planned Features

* Voice selection with preview
* Resume interrupted conversions
* ETA remaining
* Drag & Drop books
* Batch conversion
* Automatic chapter detection
* Professional user interface

## Development Philosophy

* Small changes
* Test every change
* Commit every successful feature
* Never modify working code without a backup
