# BookBuilder

Turn your books into audiobooks. BookBuilder converts **`.txt`, `.pdf`, `.docx`,
and `.epub`** files into MP3 audiobooks using natural neural voices, with a
simple desktop interface.

![version](https://img.shields.io/badge/version-1.0.0-blue)

## Features

- 📚 **Multiple formats** — txt, pdf, docx, epub
- 🎙️ **8 voices** — male and female, incl. British/European accents
- 🐢🐇 **Speed control** — 0.75× to 1.5×
- ▶️ **Voice preview** — hear a sample before converting
- 📋 **Batch queue** — add many books; they convert one after another
- 💾 **Save As** — name each book's output yourself
- 🖱️ **Drag & drop** or file picker
- 📊 **Live progress** — percent, elapsed time, and ETA

## Install

> **Requires an internet connection** — voices are generated by the Microsoft
> Edge online text-to-speech service.

### Linux

**AppImage (easiest — no install):**
1. Download `BookBuilder-<version>-x86_64.AppImage` from [Releases](../../releases).
2. Make it executable: `chmod +x BookBuilder-*.AppImage`
3. Double-click it (or run it from a terminal).

**Debian/Ubuntu (.deb):**
```bash
sudo apt install ./bookbuilder_<version>_amd64.deb
```
This adds a **BookBuilder** menu entry and auto-installs `poppler-utils` (needed
for PDF input).

### Windows
1. Download `BookBuilder-<version>-windows-setup.exe` from [Releases](../../releases).
2. Run it — it creates desktop and Start-menu shortcuts and an uninstaller.
3. *(Optional, for PDF input)* install [poppler](https://github.com/oschwartz10612/poppler-windows) and add it to PATH.

## Usage
1. **Add Book(s)** (or drag files onto the window).
2. Optionally select a book and click **Save As…** to set its output name.
3. Pick a **Voice** and **Speed**; click **▶ Play Sample** to preview.
4. Click **START QUEUE**. Audiobooks are saved to `~/BookBuilder/audiobooks/`.

## Supported formats & notes
- `.txt`, `.pdf`, `.docx`, `.epub`.
- Chapters are detected from `Chapter N` headings; books without them convert as
  a single file.
- Scanned/image-only PDFs have no text layer and cannot be read.

## Run from source (developers)
```bash
python3 main.py
```
Dependencies: `tkinterdnd2` (drag & drop, optional), `edge-tts` (TTS),
`poppler-utils` (PDF). See [installer/](installer/) for packaging.

## Build a release
```bash
python3 installer/build_release.py        # zip (current OS)
bash    installer/linux/build_deb.sh       # .deb  (Linux)
bash    installer/linux/build_appimage.sh  # AppImage (Linux)
```
Windows installers and tagged releases are built automatically by
[GitHub Actions](.github/workflows/release.yml) on a `v*` tag.
